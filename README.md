Weather Outfit ReAct Agent

A simple Python project demonstrating a ReAct (Reasoning + Acting) Agent that simulates weather conditions and suggests outfits accordingly.

📌 Overview

This project takes a city name as input, generates mock weather data, and provides outfit recommendations based on temperature and weather conditions.

It’s a beginner-friendly example to understand:

Tool usage in Python
Decision-based logic
Basic ReAct agent workflow
Features
🌍 Accepts any city as input
⛅ Simulates real-time weather
🧠 Applies reasoning to suggest outfits
👕 Simple and clear recommendations
🔁 Demonstrates ReAct agent concept
🛠️ Tech Stack
Python
Built-in libraries:
random
datetime

⚙️ How It Works
User enters a city name
Weather data is generated randomly
Temperature & condition are evaluated
Outfit suggestion is provided
